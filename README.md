## Lesson 2:
Math calculator with basic arithmetic operations without eval

```npm run calc```

Then you can calc simple math operations

Examples:

```
> 20 + 10
Result: 30
> 10 + 10 * 20 / 10 + 3
Result: 33
> 3 ^ 4
Result: 81
> 5!
Result: 120
> **4
Result 16
```
